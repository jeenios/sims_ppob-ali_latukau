# SIMS PPOB - Ali Latukau

## Deskripsi

Repository ini dibuat sebagai bagian dari **Technical Test (Assessment)** untuk posisi Web Programmer React JS di **PT Nutech Integrasi**.

Aplikasi ini adalah simulasi platform PPOB (Payment Point Online Bank) yang dibangun menggunakan React, menampilkan fitur-fitur seperti Login, Registrasi, Dashboard Saldo, dan Menu Pembayaran.

## Teknologi

- **Framework**: React.js (Vite)
- **Styling**: Tailwind CSS & Shadcn UI
- **Routing**: React Router DOM
- **Validasi**: Zod
- **State Management**: Redux Toolkit
- **API Integration**: Axios
- **etc**

## Cara Menjalankan Aplikasi

1. **Clone repository**

   ```bash
   git clone https://github.com/alilatukau/sims-ppob-alilatukau.git
   cd sims-ppob-alilatukau
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Jalankan server development**

   ```bash
   npm run dev
   ```

4. Buka browser di `http://localhost:5173`
